# 概要

Angularでルーティング時に別サイトにリダイレクトするサンプルです。

# 実行環境

* Node.js - 10.x
* Yarn - 1.17.3

# 使用ライブラリ

* Angular - 8.2.x

# 動作確認

## 1. サンプルのダウンロード

```bash
git clone git@github.com:yasu-s/ng-redirect-sample.git
```

## 2. パッケージインストール  

```bash
cd ng-redirect-sample
yarn
```

## 3. サンプルの起動  

```bash
yarn start
```
